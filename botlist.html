Lyrics Finder

Searches song lyrics using the Genius API


How to use

To invoke the bot, you will need to mention the bot

Lyrics search

~!s <query>

Searches Genius for the query and returns the first result using RichEmbed
You can also search by writing the parts of the lyrics
Treat the search command as a search engine! If the song doesn't match, try to refine the query.


Now Playing

~!np

Searches Genius for your currently playing Spotify song
The song might not match 100%


Auto Search

~!auto

Similar to Now Playing, but any time you change the song, it automatically researches the song.
Lyrics will be sent to the channel where you've invoked the bot
